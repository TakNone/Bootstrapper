# pageBlockDetails

**Description** : *A collapsible details block*

**Layer** : 214

```tl
pageBlockDetails#76768bed flags:# open:flags.0?true blocks:Vector<PageBlock> title:RichText = PageBlock;
```

---

## Parameters

| Name | Type | Description |
| :---: | :---: | :--- |
| <mark>flags</mark> | [`#`](type/#) | Flags, see TL conditional fields |
| **open** | [`flags.0?true`](type/true) | Whether the block is open by default |
| <mark>blocks</mark> | [`Vector<PageBlock>`](type/PageBlock) | Block contents |
| <mark>title</mark> | [`RichText`](type/RichText) | Always visible heading for the block |

---

## Type

[PageBlock](type/PageBlock)
