# page

**Description** : *Instant view page*

**Layer** : 216

```tl
page#98657f0d flags:# part:flags.0?true rtl:flags.1?true v2:flags.2?true url:string blocks:Vector<PageBlock> photos:Vector<Photo> documents:Vector<Document> views:flags.3?int = Page;
```

---

## Parameters

| Name | Type | Description |
| :---: | :---: | :--- |
| <mark>flags</mark> | [`#`](type/#) | Flags, see TL conditional fields |
| **part** | [`flags.0?true`](type/true) | Indicates that not full page preview is available to the client and it will need to fetch full Instant View from the server using messages.getWebPagePreview |
| **rtl** | [`flags.1?true`](type/true) | Whether the page contains RTL text |
| **v2** | [`flags.2?true`](type/true) | Whether this is an IV v2 page |
| <mark>url</mark> | [`string`](type/string) | Original page HTTP URL |
| <mark>blocks</mark> | [`Vector<PageBlock>`](type/PageBlock) | Page elements (like with HTML elements, only as TL constructors) |
| <mark>photos</mark> | [`Vector<Photo>`](type/Photo) | Photos in page |
| <mark>documents</mark> | [`Vector<Document>`](type/Document) | Media in page |
| **views** | [`flags.3?int`](type/int) | View count |

---

## Type

[Page](type/Page)
