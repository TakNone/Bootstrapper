# messages.dialogs

**Description** : *Full list of chats with messages and auxiliary data*

**Layer** : 211

```tl
messages.dialogs#15ba6c40 dialogs:Vector<Dialog> messages:Vector<Message> chats:Vector<Chat> users:Vector<User> = messages.Dialogs;
```

---

## Parameters

| Name | Type | Description |
| :---: | :---: | :--- |
| <mark>dialogs</mark> | [`Vector<Dialog>`](type/Dialog) | List of chats |
| <mark>messages</mark> | [`Vector<Message>`](type/Message) | List of last messages from each chat |
| <mark>chats</mark> | [`Vector<Chat>`](type/Chat) | List of groups mentioned in the chats |
| <mark>users</mark> | [`Vector<User>`](type/User) | List of users mentioned in messages and groups |
