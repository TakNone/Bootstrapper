# channels.channelParticipants

**Description** : *Represents multiple channel participants*

**Layer** : 218

```tl
channels.channelParticipants#9ab0feaf count:int participants:Vector<ChannelParticipant> chats:Vector<Chat> users:Vector<User> = channels.ChannelParticipants;
```

---

## Parameters

| Name | Type | Description |
| :---: | :---: | :--- |
| <mark>count</mark> | [`int`](type/int) | Total number of participants that correspond to the given query |
| <mark>participants</mark> | [`Vector<ChannelParticipant>`](type/ChannelParticipant) | Participants |
| <mark>chats</mark> | [`Vector<Chat>`](type/Chat) | Mentioned chats |
| <mark>users</mark> | [`Vector<User>`](type/User) | Users mentioned in participant info |
